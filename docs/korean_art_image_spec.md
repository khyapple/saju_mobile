# 한국적 그림 이미지 스펙 가이드

사주 모바일 앱에 들어갈 한국 전통 이미지 에셋 제작 가이드입니다.

---

## 1. 배경 패턴/텍스처 (반복 타일)

| 항목 | 권장 |
|------|------|
| 크기 | **512 x 512 px** (정사각 타일) |
| 해상도 | 72~150 DPI |
| 포맷 | **PNG** (투명 배경 필수) |
| 용량 | 50~150 KB |
| 색상 | **흰색 또는 밝은 골드 단색** (앱에서 opacity 3~7%로 깔기 때문에) |
| 비고 | 상하좌우 이음매 없이 반복 가능하게 (seamless tile) |

**배치 위치:** CosmicBackground 배경 레이어, 스플래시 화면 배경

---

## 2. 장식 일러스트 (구름, 학, 연꽃 등)

### 크기별 스펙

| 배치 위치 | 크기 | 용량 | 비고 |
|-----------|------|------|------|
| **스플래시 로고 주변** | **800 x 800 px** | ~200 KB | 정사각, 중앙 비워둘 것 (로고 자리) |
| **로그인 상단 배경** | **720 x 400 px** | ~150 KB | 상단에 걸치는 형태 |
| **프로필 목록 인사말** | **360 x 200 px** | ~80 KB | 우측에 배치, 반투명 |
| **채팅 빈 상태** | **300 x 300 px** | ~100 KB | 중앙 배치 |
| **온보딩 웰컴** | **600 x 600 px** | ~180 KB | 타이틀 뒤 배경 |

### 공통 사항

| 항목 | 권장 |
|------|------|
| 포맷 | **PNG** (투명 배경 필수) |
| 색상 | **골드(#F5C800)** 또는 **흰색** 계열 단색 권장 |
| 스타일 | 선화(line art) 또는 실루엣이 가장 잘 어울림 |

---

## 3. 구분선/테두리 장식 (단청, 문살)

| 항목 | 권장 |
|------|------|
| 크기 | **720 x 24 px** (가로 바) 또는 **360 x 12 px** (2x 타일) |
| 해상도 | 72 DPI |
| 포맷 | **PNG** (투명 배경) |
| 용량 | 10~30 KB |
| 색상 | 단청 다색(빨/파/노/초) 또는 골드 단색 |
| 비고 | 가로로 반복 가능하게 (horizontally seamless) |

**배치 위치:** DancheongBar 교체/추가, 프로필 상세 탭 구분, 글래스 카드 상단 테두리

---

## 4. 아이콘/심볼 (팔괘, 음양, 오행)

| 항목 | 권장 |
|------|------|
| 크기 | **128 x 128 px** (각 아이콘) |
| 해상도 | 72 DPI |
| 포맷 | **PNG** (투명 배경) 또는 **SVG** |
| 용량 | 5~20 KB |
| 색상 | **흰색 단색** (앱에서 tint 적용 가능) |
| 세트 | 오행 5개, 팔괘 8개, 음양 1개 등 필요한 만큼 |

**배치 위치:** 프로필 카드 아바타, 사주 차트 위젯, 궁합 분석 VS 영역, 해석 탭 섹션 아이콘

---

## 공통 주의사항

- **투명 배경 PNG** — 코스믹 배경 위에 올리기 때문에 필수
- **밝은 단색** (흰색/골드) — 어두운 배경 위에서 opacity로 조절하므로 어두운 색은 안 보임
- **선화/실루엣** 스타일 — 디테일이 너무 많으면 opacity 낮출 때 뭉개짐
- **2x, 3x 대응** — 모바일이니 위 크기의 **2배**로 그려서 축소하면 선명도 최고 (예: 512px 권장이면 1024px로 그려도 OK)
- **총 용량** — 전체 이미지 합산 **2MB 이내** 권장 (앱 크기 관리)
